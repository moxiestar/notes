**Chapters 4.1-4.4**
- Types of bonds:
	- Covalent bonds: pairs of shared valence electrons
		- Polar covalent bonds: unequally shared electrons between atoms
		- Nonpolar covalent bonds: equally shared electrons between atoms
	- ![[Screenshot 2024-09-19 at 2.09.30 PM.png]]
- Octet rule: all atoms except the smallest will move electrons until each atom has eight valence electrons
- Lewis dot symbols depict an atom's bonding capacity - the number of unpaired dots indicates the bonding capacity of that element
- Lewis structures: representation of a compound showing the connections between atoms ![[Screenshot 2024-09-19 at 2.18.31 PM.png]]
- A single bonding pair: single bond
	- Double bond, triple bond, etc
- Unshared electron pairs: lone pairs
- The number of unpaired Lewis dots indicates the bonding capacity for that element
- Elements in the same group (going down the periodic table) have the same number of valence electrons
- Drawing Lewis structures:
	- 1. Determine the number of valence electrons (count all the electrons in the atoms, add or subtract for ions)
	- 2. Arrange element symbols to show how the atoms are bonded and then connect them with single bonds (element with greatest bonding capacity = central element)
	- 3. Complete the octets of the atoms bonded to the central atom (add lone pairs till each atom has an octet)
	- 4. Check the number of valence electrons with the number calculated in Step 1
	- 5. Complete the octet on the central atom
