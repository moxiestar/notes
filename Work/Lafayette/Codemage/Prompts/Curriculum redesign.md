## **Level design**
### **Introduction: for first time users**
1. **Pitch**
	- “Have you ever wanted to build this or this or this (slick app examples) but got stuck here or here or here (extensive coding knowledge, suffering, didn’t know where to start, etc)?”
	- “You’re in the right place.”
	- “Mosaic helps you cut past all the technical difficulties and see the forest for the trees.”
2. **Placement quiz**
	- “Let’s start with a few questions.”
	- “Have you ever used AI tools like ChatGPT or Claude?”
		- “Yes”
		- “No”
		- “What’s AI?”
	- “How much do you use AI tools?”
		- “I’ve used them once or twice”
		- “I use AI regularly”
		- “I use AI every day”
		- “I build AI agents (why are you here?)”
	- “What do you use AI tools for?”
		- “School (essay writing, studying)”
		- “Work (organization, emails)”
		- “Coding/building (creating apps, writing code)”
		- “Fun (image generation, advice)”
	- “Have you written any code before?”
		- “Never”
		- “Only a tiny bit (copy-pasting, small scripts)”
		- “Yes, I know the basics (variables, loops, functions)”
		- “Yes, I am comfortable in at least one language”
	- “Have you ever used a terminal or command line interface?”
		- “No idea what that is”
		- “I've heard of it but never really used it” 
		- “I’m familiar with basic commands (cd, ls, mkdir)”
		- “I am comfortable in the terminal”
	- “What are you hoping to get out of Mosaic?”
		- “Learn how to code effectively with AI”
		- “Get comfortable working in the terminal”
		- “Build better apps/projects”
		- “Have fun working with AI”
	- **Levels for placement**
		- Beginner
		- Intermediate
		- Advanced
3. **Sign-up page**
4. **Interface tutorial**
	- Introduction to the interface via the tamagotchi pet
