## **Reddit post (r/SideProject)**
**I shipped a coding education platform today — here's what building it solo during the semester actually taught me**

I spent the last few months building a coding education platform as a side project while studying at UNC. Shipping it today — wanted to share the journey here.

The product is called CodeMage. It teaches high schoolers to work with AI tools — Claude, Claude Code — through real challenges instead of tutorials. 12 levels, a Socratic AI tutor, and a certificate on completion.

The build has been a real education in itself. Some things I learned the hard way:
- Stripe webhooks will silently swallow errors if your canonical domain redirects (codemage.dev 307s to www.codemage.dev — Stripe was POSTing to the non-www URL and the body never made it through)
- Supabase type generation gets stale fast once you start adding tables mid-build
- The hardest part wasn't the code. It was deciding what the first three free levels should teach — balancing "accessible enough for a complete beginner" against "real enough to be worth finishing"

The stack is Next.js 14, Supabase, Anthropic API, Stripe, Vercel. Built solo throughout this semester alongside coursework.

Launching into the Triangle NC market first since that's home turf — NC has a new CS graduation requirement taking effect next school year which creates real demand.

First 3 levels are free with no sign-up. Happy to send a full-access code to anyone who wants to poke around under the hood — just comment or DM.

Codemage.dev — would love any feedback from other builders.
